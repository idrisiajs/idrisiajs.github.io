# IDRISIAJS Website v2

## Paste the API key
Open `config.js` and replace `PASTE_YOUR_API_KEY_HERE` with your restricted YouTube Data API v3 key. Keep the quotation marks.

## Google Cloud restrictions
Websites: `https://idrisiajs.com/*`, `https://www.idrisiajs.com/*`, `https://idrisiajs.github.io/*`
API: YouTube Data API v3 only.

## Automatic content
Latest Videos loads automatically from the channel upload feed. Shorts loads from the public playlist ID in `SHORTS_PLAYLIST_ID`. Add each Short to that playlist.

## Upload
Replace the old website files with everything in this ZIP and commit to the main branch.
